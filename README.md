# Secure Password Manager

A desktop-based Password Manager built with Python, CustomTkinter, SQLite, and Cryptography. This application allows users to securely store, manage, and retrieve website credentials using a master password and encrypted password storage.

## Features

* Master Password Authentication
* Secure Password Storage
* Password Encryption using Fernet
* Search Saved Credentials
* Copy Passwords to Clipboard
* Delete Stored Credentials
* User-Friendly Graphical Interface
* Local SQLite Database Storage

## Technologies Used

* Python
* CustomTkinter
* SQLite
* Cryptography (Fernet)
* Hashlib
* PBKDF2 Key Derivation

## Security Features

* Master password authentication system
* Password hashing with SHA-256
* Salt generation for improved security
* Key derivation using PBKDF2
* Encryption and decryption using Fernet
* Local credential storage using SQLite
